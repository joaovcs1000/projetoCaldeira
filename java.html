A última parte do curso de programação da Alura contempla a linguagem Java. Começamos o curso com o aprendizado da sintaxe básica do java, aprendendo, de certa forma, a lógica de programação novamente. Assim que acabamos a parte básica, iniciamos o assunto de orientação de objetos e seus benefícios para a programação na linguagem. Por último, ingressamos no mundo do polimorfismo, com esse conhecimento conseguimos entender melhor os objetivos e funções da orientação a objetos. O projeto final do curso de java é um banco, utilizando todas as funcionalidades aprendidas!
